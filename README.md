## Installation

Install dependencies:

```bash
npm install
```

## Running Tests

Run all tests:

```bash
npm test
```

### Run Specific Projects

Run tests on Chromium:

```bash
npm run test:chrome
```

Run tests on Firefox:

```bash
npm run test:firefox
```

### Run Modes

Run tests in headed mode (visible browser):

```bash
npm run test:headed
```

Run tests in debug mode:

```bash
npm run test:debug
```

### Specific Configurations

Run tests with 1920x1080 resolution:

```bash
npm run test:1920
```

Run tests with 1366x768 resolution:

```bash
npm run test:1366
```

Run tests with 4 parallel workers:

```bash
npm run test:workers
```

## Reporting

Show the test report:

```bash
npm run report
```
